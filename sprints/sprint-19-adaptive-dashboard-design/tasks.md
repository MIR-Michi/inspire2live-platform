| S19-T08 | **Tile sizing, density, presets, and focus mode.** Implement controlled Compact/Standard/Wide sizes, content-level size constraints, Balanced/Focus/Overview presets, temporary supporting-rail focus collapse, and predictable mobile mapping. | TBD | Not Started | Structured responsive grid only; no absolute pixel positioning or overlap. |
| S19-T09 | **Migrate Communications dashboards.** Move Communications My and Team dashboards to the shared registry/shell. | S19-T10 | **Migrate Admin and role dashboards.** Move Admin plus coordinator/advocate/clinician/researcher dashboard variants to the shared system. Preserve view toggles, effective-viewer behavior, required admin health content, role-specific queries, and deep links. | TBD | Not Started | Required governance/health tiles can move but cannot be hidden. |
| S19-T11 | **Motion system and reduced-motion contract.** Define shared durations/easing and primitives for hover/press, expand/collapse, drag lift/drop, tile insertion/removal, filter changes, autosave, loading skeletons, and new-data highlights. | S19-T12 | **Task completion celebration.** Wire a direct user-triggered completion event to success-check feedback, short row highlight, localized confetti, and smooth removal when finished tasks are hidden.
